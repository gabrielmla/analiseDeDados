# analiseDeDados
Analises feitas para disciplina de analisde de dados 2.

* Ao utilizar modelo de regressão, quanto menor o p-value de uma variável, menor a probabilidade da correlação dela com a variável resposta ser aleatória.
* Bom erro no treino não significa bom erro no teste.
* Testar o lambda nos dados de validação. Ele se aproxima mais do erro encontrado nos dados de teste.
* Minimizar RSS e a magnitude dos coeficientes, controlando o overfiting do modelo ( Diminuir BIAS e aumentar Generalização, achar um equilibrio ).
* Algoritmo dos vizinhos gera uma boa interpolação caso os dados sejam densos, nesse metodo vejo os vizinhos mais próximos.
* Na regressão com Kernel, em vez de calcular distância a vizinhos, calcula para todos os pontos. Quando a distância passar de um limite de tolerância, o peso será zero. Se o limite de tolerância incluir todos os pontos, o modelo gerado será a média.
